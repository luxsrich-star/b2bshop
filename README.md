# B2B Shop — Мульти-магазин

Оптовая B2B платформа с поддержкой нескольких магазинов, каждый со своими ценами.

---

## Структура

```
/admin              — Супер-админ (управление магазинами + общий каталог)
/admin/[slug]       — Панель продавца (цены, заказы, настройки)
/shop/[slug]        — Витрина для клиентов
```

---

## Запуск локально

### 1. Установить Node.js
Скачай и установи [Node.js 18+](https://nodejs.org/)

### 2. Распаковать проект
Положи папку `b2bshop` куда удобно.

### 3. Установить зависимости
```bash
cd b2bshop
npm install
```

### 4. Создать .env.local
Создай файл `.env.local` в корне проекта:
```
SUPER_ADMIN_PASSWORD=твой_пароль
```

### 5. Запустить
```bash
npm run dev
```

Открой http://localhost:3000

## Деплой на Railway

### 1. Залить на GitHub
```bash
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/твой-юзер/b2bshop.git
git push -u origin main
```

### 2. Создать проект на Railway
- Зайди на [railway.app](https://railway.app)
- New Project → Deploy from GitHub → выбери репозиторий
- Railway сам определит Next.js и задеплоит

### 3. Добавить переменную окружения
В Railway → твой проект → Variables:
```
SUPER_ADMIN_PASSWORD = твой_секретный_пароль
```

### 4. Persistent Storage (важно!)
Папка `data/` и `public/uploads/` должны сохраняться между деплоями.
На Railway: Add Service → Volume → примонтируй к `/app/data` и `/app/public/uploads`

## Как пользоваться

### Супер-админ `/admin`
- Пароль из `SUPER_ADMIN_PASSWORD` (по умолчанию: `superadmin`)
- Создаёшь магазины: название, slug, пароль
- Управляешь общим каталогом: категории и товары с базовыми ценами

### Панель продавца `/admin/[slug]`
- Вводишь пароль конкретного магазина
- **Цены и остатки** — задаёшь свою цену и остаток для каждого товара
- **Заказы** — видишь заказы только своего магазина, меняешь статусы, печатаешь чеки
- **Настройки** — название, логотип, WhatsApp

### Витрина `/shop/[slug]`
- Клиент открывает ссылку, видит товары с ценами конкретного магазина
- Добавляет в корзину, оформляет заказ (имя + Telegram)
- Получает печатный чек, может поделиться в Telegram
